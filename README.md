Currency Exchange Service

This project is aimed at developing a currency exchange service that provides and stores current currency exchange rates as well as their change history.

Installation Clone the repository: git clone https://github.com/Mitos35/currency-rate.git

Navigate to the project directory: cd currency-rate-app

Build the project using Gradle: ./gradlew buil Run the application: java -jar build/libs/currency-rate.jar

Optionally, run the application along with PostgreSQL using Docker: docker-compose up -d
